About wartwork

About

wartwork is a wargame artwork blog from John Edwards, the publisher of https://www.simpubs.org, the archive of SPI games.

In re-creating classic artwork of the 70s and 80s, mainly from SPI, GDW and The Avalon Hill Games Company (aka AH, or TAHGC), I have learned a lot about design principles and practices. This site is intended to bring some together and publish them to share with like minded enthusiasts/fools.

The site will focus on counter art, but will digress to maps, cover and rules/docs too. I have developed an unhealthy interest in fonts, so there is a fonts subsection. Vassal is a (brilliant) tool to play board games on computer. There is likely to be a bunch of tech content, as I am an IT guy, and love technology. Finally, we might share some favo(u)rite things.

All in all, content will be tagged with these labels, which will include

  • counters
  • maps
  • covers
  • docs (rules, charts, other publications)
  • fonts
  • vassal
  • tech
  • favs (a few of our favorite things)

Click TAGS to see content by these subjects.

Tech

wartwork is created with Hugo, a static (web) site generator, or SSG. It is based on the ghostwriter theme, with some minimal changes. It will be hosted on github (https://jzedwards.github.io) and a couple of free web sites as backup too (X10 and Infinity Hosting). It is saved on github at jzedwards/jzhugo, and I hope to automate the deployment on github pages very soon.
